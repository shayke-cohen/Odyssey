Ask for the redesign proposal and what problem it solves compared to the current design if not provided.
UX Reviewer scores user impact: does the redesign reduce friction, clarify hierarchy, and handle edge states better?
Engineer estimates build delta — net effort change versus the current implementation.
Pattern Reviewer lists every component or token divergence introduced and whether each is justified.
Output: a single consolidated feedback document with three sections, a go/no-go verdict, and required changes before approval.
