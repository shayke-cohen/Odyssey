name: "Accessibility design review"
sortOrder: 5
---

Ask for the design file or description and the target platform (web/iOS/Android) if not provided.
Accessibility Reviewer audits four dimensions in order: contrast ratios, focus/tab order, screen-reader label coverage, and touch-target sizes.
Each dimension gets a pass/fail verdict with specific element callouts and the relevant WCAG criterion.
Designer addresses each failure; Reviewer re-checks only the changed elements and approves or re-flags.
Output: annotated design with a four-dimension audit table and a final pass/fail summary per dimension.
