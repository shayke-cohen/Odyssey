Kickoff Goal
Prepare a design review thread for the new onboarding UX and align on the top 3 changes.
`Recent` stays fast for repeat users. `All` stays visible for discovery. `Cards / List` lets people browse the way they prefer.

Why I Like This Direction

  • One creation flow instead of separate agent/group dialogs.
  • Agents and groups stay fully discoverable inside the same sheet.
  • The toggle gives density choice without forcing one browsing style on everyone.
  • We can default to `List` for clarity and let power users switch to `Cards` when they want faster visual scanning.
If this feels right, the next product decision is simple: should the real app default to `List` or remember the user’s last picker preference?
