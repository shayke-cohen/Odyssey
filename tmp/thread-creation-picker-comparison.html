Recommendation

  • Keep the overall product language from today so the app still feels familiar.
  • Use the list/card hybrid for `All Agents` and `All Groups` inside the unified create sheet.
  • Keep Recent as quick chips to preserve speed for repeat users.
  • If we want denser scanning later, we can make the list rows slightly more compact without returning to the fully ambiguous small-card grid.
My view: the unified flow should not use the exact current picker. It should reuse today’s content, but present it in a more decision-friendly structure.
